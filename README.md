## Installation & Setup

### Requirements
- PHP 7.0 or higher
- Web server (Apache, Nginx, etc.)
- Modern web browser

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/yourusername/GORABROTHER-CASINO.git
cd GORABROTHER-CASINO
```

2. Start a local PHP server:
```bash
php -S localhost:8000
```

3. Open your browser and navigate to:
```
http://localhost:8000
```

### Deployment

1. Upload all files to your web hosting server
2. Ensure PHP is enabled on your server
3. Update file permissions if necessary
4. Access your site via your domain name
